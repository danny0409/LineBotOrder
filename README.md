# LineBotOrder

## 訂便當 BOT
由事先建立店家資訊，店名、連絡電話、菜單…等資訊。
使用角色分為發起人與參與者

## 使用方法
menu 或 menu_text 叫出主功能表

menu : 圖形按鈕選單 (電腦版無)
menu_text：純文字互動選單
